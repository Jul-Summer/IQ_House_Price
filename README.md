# HousePrice 

**Цель проекта:** разработать сервис для оперативного использования модели предсказания цен недвижимости агентами и аналитиками.

HousePrice — это инструмент для прогнозирования стоимости домов с использованием методов машинного обучения. Сервис позволяет быстро оценивать объекты недвижимости на основе их характеристик и местоположения.

---

###  Обзор проекта

Проект состоит из основных частей:

1. **EDA, предобработка и визуализация данных**  
   Исследование структуры данных, работа с пропущенными значениями и выбросами, визуализация распределений и взаимосвязей признаков.

2. **Обучение моделей**  
   Построение моделей машинного обучения (Random Forest и XGBoost с оптимизацией через Optuna) на очищенных и предобработанных данных. Оценка качества по метрикам MSE, RMSE, MAE и R².



### Структура проекта

#### Данные и ноутбуки
- `HousePrice_EDA.ipynb` – анализ структуры датасета, визуализация признаков, работа с пропущенными значениями и выбросами.  
- `HousePrice_Baseline_v.ipynb` – базовое обучение модели, подготовка начальных предсказаний и сохранение очищенного датасета.  
- `HousePrice_XGB_Optuna_v.ipynb` – обучение XGBoost модели с подбором гиперпараметров через Optuna.  

- `data/data.csv` – исходный датасет.  
- `data/Preprocessed_Data.csv` – предобработанный датасет, сохранённый для использования в моделях.
